## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/ChatMeNow.git`
3. Create a feature branch: `git checkout -b feature/amazing-feature`
4. Make your changes
5. Test your changes: `./test-api.sh`
6. Commit: `git commit -m 'Add amazing feature'`
7. Push: `git push origin feature/amazing-feature`
8. Open a Pull Request

## Development Guidelines

### Code Style

**TypeScript/NestJS:**

- Use ESLint with provided config
- Follow NestJS best practices
- Use DTOs for validation
- Add JSDoc comments for public APIs

**Go:**

- Use `gofmt` for formatting
- Follow Go best practices
- Add godoc comments
- Keep functions small and focused

### Commit Messages

Use conventional commits:

- `feat: add new feature`
- `fix: bug fix`
- `docs: documentation changes`
- `refactor: code refactoring`
- `test: add tests`
- `chore: maintenance tasks`

### Testing

- Write tests for new features
- Ensure existing tests pass
- Test manually with provided scripts

### Pull Request Process

1. Update README.md with details of changes if needed
2. Update DEVELOPMENT.md if adding new dev workflows
3. Ensure all tests pass
4. Request review from maintainers
